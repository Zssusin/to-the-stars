## 植入物系统

### 概述

> "良子盯着天花板发呆...习惯性地查询了自己的内置时计。现在是10:11:23。这条信息进入了她的意识。"

**基本构成**：
| 类型 | 功能 |
|------|------|
| 神经连接芯片 | 连接大脑和感觉系统 |
| 处理节点 | 本地计算能力 |
| I/O设备 | 连接互联网 |
| 肌肉强化结构 | 增强体能(通常休眠) |
| 心肺支持结构 | 维持生命功能 |
| 纳米机器 | 免疫、癌症预防、医学永生 |

### 能量与维护

> "植入物花掉的额外能量比人类身体正常运转要多很多。良子从蛋糕中摄入的卡路里大部分都为此储存了起来。"

- 原材料和能源从食物中积累
- 纳米机器从母体进入胚胎（先天配置）
- 绝对可靠，失效都有情有可原的原因
- 无需定期去维护设施

### 个人卫生

> "大概靠植入物吧，虽然不洗澡感觉怪怪的。"

- 基本卫生由植入物处理
- 智能头发自动清洁、维持长度
- 发型可自动调整

### 军事改造

| 平民植入物 | 军事改造 |
|------------|----------|
| 肌肉强化(休眠) | 骨骼强化 |
| 基础处理节点 | 感觉强化(光学) |
| 标准I/O | 进一步肌肉强化 |
| - | 重要器官改造 |
| - | 处理节点大幅扩展 |

> "这些改造并不需要手术——只需要不同的纳米机器、软件指令和原料进料流。"

## 虚拟现实

### 政策限制

> "执政体更希望他们张开嘴交流...用于启动内置VR芯片的计时券则被定成了让人难以想象的高价。"

**限制原因**：
- 防止人类永久沉浸虚拟世界
- 维持人类生存现状的决心
- 鼓励现实社交

### 军事应用

> "现代军事最革新的一面之一是大量、近乎全面地将VR技术用作训练。几乎所有传统训练设备都被VR代替了。"

**优势**：
- 节省资源和时间
- 加速主观时间（比现实更快）
- 武器训练、野战演习、课堂教学

### 日常使用场景

| 场景 | 描述 |
|------|------|
| 会议 | 远程VR参加，可自定义环境 |
| 游戏 | 第一人称多人对战，需VR使用费 |
| 约会 | "里面真的做什么都可以" |
| 娱乐 | 虚拟网球场、阅读间、隐私室 |
| **电影院** | 支持VR接口、气味模拟、重力调整、脑内音频直连 |

### 民间vs军方

| 平民 | 军方/魔法少女 |
|------|---------------|
| 高昂使用费 | 免费使用 |
| 速度限制 | 加速时间训练 |
| 功能限制 | 完整功能 |

### 魔法少女住房选择 (from "Emma's Story 3")
| 住房类型 | 描述 |
|----------|------|
| **MSY Housing** | 标准魔法少女公寓 |
| **希望教宿舍** | 教堂附设公寓，可租用单间或双人间 |
| **家庭住宅** | 与家人同住（未成年魔法少女常见） |

**宿舍政策**：
- 新契约的魔法少女建议合住，便于训练配合
- 年轻魔法少女配对时，系统自动匹配性格相合的室友
- 艾玛选择希望教宿舍，获得独立的高效型公寓 (efficiency-style apartment)


### 老祖宗偏好

> "老祖宗们讨论正经事时更喜欢呆在会议室里...年轻人或许会选个能放松的地方...但老祖宗们似乎总觉得一张公司会议室的会议桌才最适合紧张的局势。"

**限制与放开**：
- 平民：故意降低速度
- 军方：无限制
- 魔法少女：天然能力

- 可建立私人信道（两人对话）
- 可建立群组信道（团队通讯）
- 可传输图像（通过读心者）
- 语速可超过阅读速度

- 不重要邮件自动回复
- AI回复有小标记区分

## 社会生活

### 后稀缺经济

> "人类已经到达了历史的尽头，终于开始建设一个完美社会...但她并不觉得这有什么完美的。"

- 几乎所有物质需求都能满足
- 生产性工作是志愿的
- 战时开始削减配给

### 地球在接触战争期间的地位
在长达数十年的接触战争（Contact War）中，地球母星从未直接遭受过外星攻击。这使得地球成为了整个人类文明的避风港和物资极其丰裕的核心区域。
- **奢侈生活**：由于地球设施完好无损，平民依然享受着各种便利设施。
- **配给制 (Alloctons)**：虽然处于战时，但只要每月还有配给额度（allocs），居民就可以自由享受娱乐生活。如果配给耗尽，通常只需等待下个月即可恢复。
- **安全感**：地球居民对星际前线的残酷战争感知相对较低，生活基调依然偏向和平。

### 教育

- 学校教育+植入物辅助
- 信息直接通过芯片"塞进思考"
- 视觉材料在眼前飞快闪过
- 可进入子课题纵深阅读

### 宗教与文化

> "希望教...唯一仍有新信徒加入的宗教"

- 一元文化：全球高度同质化
- 希望教是主要宗教

### 医疗永生
