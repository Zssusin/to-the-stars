## 地面战思想

### 核心原则

> "当前通行的地面战思想着重强调机动性、灵活性、还有存续性。"

| 原则 | 描述 |
|------|------|
| **机动性** | 快速移动是关键 |
| **灵活性** | 避开据点，打击弱点 |
| **存续性** | 指挥系统抗打击 |

### 战术要点

> "胜利都属于最先打中对手弱点的那一方。"

- 隐藏弱点成为不可能（双方都有战场观察力）
- 真正的纵深防御通常无法建立（轨道炮火威胁）
- 面对进攻→抢先打回去
- 应付奇袭→伺机反击
- 消灭指挥系统比摧毁物资更有效

### 指挥系统三大功能

| 功能 | 描述 |
|------|------|
| **信息分发** | 决定谁需要收到什么信息 |
| **战术指挥** | 提供高层次作战目标 |
| **统一意见** | 在信息不足时做出统一判断 |

> "真正的命令往往只是一个简要的作战目标。"

### 通讯是生命线

> "同时失去指挥和通讯就相当危险了。地面作战的核心思想就是要同时破坏敌人部队的两个要素。"

## 魔法少女战术

### 战场定位

> "每一名魔法少女的作战能力都至少相当于一支连队，也有很多施法者达到过营级乃至团级的作战表现。"

**核心教训**：
- 落单的施法者容易被各个击破
- 集中太多则成为超级武器目标
- 解决方案：担任军官，依靠士兵支援

### 编队原则

> "除非绝对必要，否则魔法少女决不能单独作战。"

| 编制 | 人数 | 说明 |
|------|------|------|
| 最小分组 | 5人 | 先头部队最低限制 |
| 标准分组 | 12人 | 一般行动 |
| 大规模 | 50人 | 整个团一起行动 |

### 魔女师

> "像二战早期的装甲部队...把魔法少女集中到专门的魔女师里。"

- 担任进攻矛头
- 魔女突击兵：牺牲防御换突击冲力
- 技术法师：非战斗专长，避开激烈战斗

### 魔女优先政策

> "一名魔法少女的生命价值远远高于一般士兵...哪怕要把手下的整个连队派去送死，也得优先保全自己的性命。"

**双向教育**：
- 魔法少女：放弃英雄主义
- 普通士兵：保护少女长官如战旗

> "无数的士兵和连队不惜违背命令，自发地牺牲着自己的生命，只为了让他们的少女长官能够最后活下来。"

### 飞行能力

> "在瘴气之外的飞行能力...很困难。真正的飞行少女是很罕见的。"

- 所有人都能做简单的空中变向
- 连续飞行需要大量魔力
- 瘴气内飞行容易很多
- 晓美焰刚契约就能飞（特殊）

## 魔兽狩猎

### 狩猎专业化

> "行会对整个魔法少女系统进行的改进中最有价值的一个就是魔兽狩猎过程的专业化。"

**效果**：
- 死亡率直线下降
- 悲叹立方从持久不足→持久盈余
- 大多数成员解放出来从事其他工作

### 核心思想

| 原则 | 实现 |
|------|------|
| **效率** | 严格统计规律的巡逻 |
| **安全** | 模板化战术、任务分配 |
| **救援** | 平民救援放在重要位置 |

### 标准战术：传送营救法

> "从图像里找到安全落点之后马上传送到被害者身边，依次救出每一个人，最后传送回来。"

**人员配置**：
- 传送者（必须）
- 结界师（必须）
- 透视眼（推荐）
- 读心者（推荐）
- 隐形师（可选，稀缺）

### 标准战术：轰炸编队

> "传送打击编队的俗称。一组拥有范围魔法的人在战场之上快速来回，用打了就跑的方式打击敌人。"

**分工**：
| 角色 | 职责 |
|------|------|
| 传送者 | 提供机动性 |
| 结界师 | 防御流弹 |
| 范围法师 | 输出伤害 |
| 透视读心 | 传送辅助（可选） |

**战术流程**：
1. 传送到目标地点
2. 结界师防护
3. 范围法师输出
4. 传送离开

### 瘴气特性

> "瘴气几乎可以说是一种特别的生物...会不断进化，发展出更加复杂的手段逃避监控。"

- 可伪造行人数字签名
- 可伪造摄像头画面
- 地域性差异（跨星球尺度）

## 太空作战

### 舰队构成

| 人类 | 外星人(章鱼) |
|------|--------------|
| 战列舰为主力 | 航母为主力 |
| 少量轻型航母 | 大量战斗机轰炸机 |
| 隐形轻型巡防舰 | 闪现炮 |
| 魔法少女部队 | 远程攻击 |

> "每一艘有人驾驶的人类船只都是魔法少女部队的移动基地，配有医疗设施、居住空间、悲叹立方和特制武器。"

### 钻头阵

> "战列舰都布置在最尖端，前方集中巡洋舰和巡防舰形成进攻矛头。"

- V形阵势给战列舰让开火力线
- 舍曼主炮沿战舰脊线安装，只能向前
- 矛尖位置由精锐舰队担任

### 完全指挥模式

> "只有舰队中的高级军官才能有幸体验...那种成为伟大集团一份子的感觉。"

- 借助多支舰队的传感器感知
- 各种电磁辐射光谱
- 超光速引擎的颤动
- 时空间的舒展

### 魔法少女太空战

> "轨道上驻扎的外星战舰观察到了一个不可思议的现象——几万个穿着太空服的人类漂浮在轨道上，向它们发起了冲锋。"

**角色**：
- 隐形巡防舰搭载魔女小队
- 太空战的"击坠王"
- 传送者价值极高

### 敌对单位：收割者 (Reapers)
- **描述**：章鱼 (Squid) 的巨型战斗站。
- **规模**：体积相当于人类巡洋舰，拥有护卫舰级别的护盾强度。
- **战力**：火力巨大，能在数英里外对人类阵位造成毁灭性打击。

## 军事编制

### 地面军队

| 单位 | 备注 |
|------|------|
| 班 | 基础单位 |
| 排 | 配魔法少女排长 |
| 连 | 配魔法少女连长 |
| 营 | 多个连 |
| 团 | 约50名魔法少女 |
| 师 | 魔女师为精锐 |
| **魔女兵团 (Magi Oplon)** | 指魔法少女所属的特种部队编制（如第 5 魔女兵团），通常作为军队的进攻核心。 |
| **冲锋营 (Shock Battalion)** | 魔法少女部队中的精锐突击力量，负责正面硬碰硬的重火力突击。 |
| **MagOps (魔法作战小队)** | 高度专业化的特种作战小队，负责补给线骚扰、救援、暗杀等任务。成员往往是数十年经验的资深魔法少女（如 97 岁的 Nodoka）。在《艾玛的故事2》中，MagOps Team Four 执行萨姆萨拉补给线打击任务并救出艾玛。 |
| **灵魂卫队 (Soul Guard)** | 专业化训练路径之一。在《艾玛的故事3》中，艾玛选择加入灵魂卫队而非轻步兵学校 (Light Infantry School)，后者更为苛刻。Alanis 曾尝试 LIS 但中途退出。 |
| **轻步兵学校 (LIS)** | 更严苛的训练路径，不给魔法少女特殊待遇。适合想要从事空中突击 (Aerial Assault) 等极端专业的人。 |

### MSY 职业教育 (MSY Careers and Education) (from "Emma's Story 3")
为魔法少女提供课程选择和职业规划的官方机构：
- **顾问**：如金英美 (Kim Young-mi)，曾参与最早期对章鱼作战
- **新人课程**：战前训练序列，专注魔兽狩猎、能力开发、自我修复、灵魂宝石管理
- **专业化**：根据能力类别（如召唤师）提供专精发展路径
- **服役期**：魔法少女征召期为 30 年
- **建议**：顾问建议优先完成魔法训练，再追求民用教育

## 战斗描写参考

### 魔兽战斗示例

**轰炸编队攻击**：
```
黄衣女孩微微浮空，在周身聚起了一个闪电球。球体慢慢变大，
将另外几人也包在了里面...电流直接穿过了她的身体，没有造成
任何影响。接着就是突如其来的爆炸，冲击波把周围的魔兽群烧
成了飞灰。
```

**结界师防护**：
```
什么东西猛地撞上了结界顶部，原本透亮的蓝色密实起来，
迸出火花，发出了尖锐的摩擦声。

接着她伸手虚虚一推，结界向外猛撞，推开了近处站着的
几只魔兽，其中大多数就此气化消灭。
```

**传送战术**：
```
良子集中精神，驱动魔力向远方伸展开去——
然后她们就到了那里。周围的结界闪烁着通透的蓝光，
将恰在附近的两只魔兽切成了四半。
```

### 受伤描写

> "真正让良子难以忘怀的还是她那齐肘而断的左臂...她才发现她已经少了一只眼睛，连带着脸上也缺了好大一块。"

**处理方式**：
- 悲叹立方贴上黑色宝石
- 送医院（传送者可200公里距离）
- 自行施加麻醉
- "对你们来说，这根本不算什么"

### 太空战描写

**舰队交火**：
```
炫目的白色猛烈地挤占了整个图像，就算是在眼部增强装置
和层层虚拟现实的过滤之下，参谋部的成员们依然下意识地
抖了一抖。接着图像就消失了。
```

**闪现炮攻击**：
```
原本无懈可击的地点上凭空出现了外星人的核弹和反物质炸弹。
炸弹随即爆开，在各种关键位置造成了令人痛心的损失。
```

## 写作技巧

### 战斗节奏

- **短句**营造紧张感
- 穿插**念话**保持信息流通
- 战斗间隙进行**战术分析**
- 使用**新手视角**解释能力机制
- 强调**直觉与本能**（如传送者本能地出现在敌人背后）

### 魔法表现

| 能力 | 视觉效果 |
|------|----------|
| 传送 | 扭曲感、穿刺感 |
| 结界 | 透亮蓝色、密实、火花 |
| 闪电 | 闪电球、冲击波 |
| 念力 | 青色鬼手 |
| 召唤 | 大炮、机器人 |

### 团队协作要素

- 读心者做领队指挥
- 结界师保护脆弱成员
- 传送者提供机动性
- 范围法师负责输出
- 透视眼提供情报

- 透视眼提供情报

### 特殊装备与能力

**灵魂宝石护甲**：
- **外观**：液态金属包裹灵魂宝石，变为透明/保护层。
- **功能**：保护由于不连接身体而变得脆弱的灵魂宝石（本体）。
- **来源**：杏子等资深魔法少女的私藏高科技装备。

## 《艾玛的故事》战术参考

### 魔法少女协同战术

| 战术名称 | 配合者 | 描述 |
|----------|--------|------|
| **火风龙卷** | 艾玛 + 蕾妮 | 蕾妮在周围形成火幕，艾玛用风暴包围火幕，形成向外扩展的白色毁灭锥体，焚烧并熔化无高密度护盾的敌人 |
| **投掷突击** | 绫那 + 艾玛 | 绫那将艾玛高速投掷向敌方阵地，艾玛利用冲击动能砸穿目标（模拟训练中曾消灭整支装甲部队） |
| **邓杰内斯 (Dungeness)** | 艾玛 + 艾米莉 + 秦朵 | 以螃蟹命名：艾玛作"颚"近战撕裂；艾米莉作"钳"用水系魔法削弱护盾；秦朵作"壳"用镜像防护；阿拉尼斯狙击掩护 |

### 战斗描写示例（艾玛的故事）

**火风龙卷战术**：
```
"现在！"蕾妮喊道，在原地旋转，在她周围形成一道火幕。
片刻后，一股旋转的风暴包围了火幕，把它变成一个向外迅速
扩展的火焰龙卷风，吞没了突击兵。
```

**漩涡球攻击**：
```
艾玛高速从天而降扑向目标。漩涡球几乎立刻发射出去，
在坦克前方爆炸，使这辆几十吨的车辆略微偏转，同时
扬起一阵尘土。
```

**邓杰内斯战术**：
```
一排排带着倒影的镜像出现在她们周围。艾米莉带头进攻，
把葫芦里的水呈弧形泼洒出去，撞击着章鱼的护盾直到护盾
失效，然后将它们勒死、溺死或用其他方式杀死。艾玛帮忙
标记发现的敌人，协同配合砍杀冲入敌阵。
```
