## 灵魂宝石基础

### 本质
- 魔法少女的"真正身体"是灵魂宝石，肉体只是容器
- 灵魂宝石承载着魔法少女的意识、记忆和魔力

### 位置（作者回答）
| 角色 | 灵魂宝石位置 |
|------|--------------|
| 亚纱美 | 心脏附近 |
| 克莱丽丝·凡·罗萨姆 | 额头 |
| 黑井奈奈 | 喉咙处 |
| 帕特里西亚 | 额头 |
| 岸田麻希 | 手上 |
| 浅香 | 腹部 |

### 污染机制
- **污染来源**：负面情绪、使用魔法、战斗消耗
- **净化方式**：使用悲叹立方吸收污染
- **危险阈值**：污染过度会导致灵魂宝石完全黑化

### 距离限制
- 原作设定：灵魂宝石离身体超过 100 米会失去对身体的控制。
- TTS 中此设定应该保留，但未明确提及。

## TTS世界观重大变化

### 悲叹立方替代悲叹之种

TTS世界中，原作的"悲叹之种"(Grief Seed) 被"悲叹立方"(Grief Cube) 取代。这一变化对MSY的发展具有深远影响：

| 变化 | 影响 |
|------|------|
| **体积更小** | 悲叹立方比悲叹之种小得多，分配更容易公平化，减少了欺骗和内讧 |
| **合作促进** | 类似"公司分红"而非"争夺一张大额支票"，团队合作更自然 |
| **没有使魔成长** | 不存在"使魔吃人长成魔女"的机制，消除了杏子vs沙耶加第5-7集式的道德争论 |
| **没有魔女化** | 不再有少女变魔女的悲剧，消除了"真相揭露"导致团队崩溃的风险 |

### 这些变化为何重要

作者原话：
> "提个不相关的，我想评论一下新的魔兽系统和旧有的魔女系统的关键区别，这使得类似MSY等组织的发展成为可能，至少在促进合作和团结方面是这样。"

## 魔女与圆环之理

### TTS中的魔女
- 由于小圆的愿望重塑世界，魔女系统被魔兽取代
- 但魔女作为概念仍然存在于历史和部分角色的记忆中

### 圆环之理 (Law of Cycles)
- 小圆成为女神后创造的规则
- **作用**：接引陷入绝望的魔法少女，阻止她们变成魔女
- **范围**：不仅接引"黑化"的魔法少女，也包括在战斗中死亡的
- 作者确认：所有死去的魔法少女都会被圆环之理接引

### 魔女之夜 (Walpurgisnacht)
- 作者秘密设定：**魔女之夜是一个穿越时空的魔女**
- 这解释了她在历史上多次造成大灾难的原因
- 暗示：克莱丽丝·凡·罗萨姆与魔女之夜有关联

### 女神的能力与限制 (from "Emma's Story 2")
《艾玛的故事 2》第 1 章开头描述了女神（圆）拯救魔法少女后的状态：

| 特性 | 描述 |
|------|------|
| **疲惫** | 女神在"撕开时空裂隙"拯救魔法少女后会感到疲惫，需要吃东西恢复。 |
| **非全能** | 沙耶香明确说"你不是完全全能的"（You're not completely omnipotent）。 |
| **牺牲** | 拯救每个魔法少女都消耗能量，但圆坚持"每一个女孩都重要"。 |
| **历史干预** | 沙耶香提到曾看到圆救过"圣女贞德"（Joan of Arc）。 |

> "Every girl is important, Sayaka." —— 圆在沙耶香质疑是否需要救艾玛时冷冷说道，房间物理性降温。

## 悲叹立方系统

### 获取方式
- 击杀魔兽获得悲叹立方
- MSY统一管理和分配悲叹立方

### 经济系统
- 悲叹立方在MSY内部具有货币功能
- 可用于交易物品、服务
- 有投机悲叹立方价格的可能（作者提及）

### MSY悲叹立方管理
根据MSY初始章程第一条：
> "建立集体管理的悲叹之种保险库，从而减轻任一小队可能出现的盈亏波动。每个小队都应当贡献其占收入特定比例的悲叹之种，或固定的最低额度，取其中较高者。"

## 克隆复活项目

### 时间线
- **2350-2380**：开始实施MSY克隆项目
- **2395-2420**：首次成功利用克隆作为魔法少女的新身体

### 机制
- 魔法少女死亡后，如果灵魂宝石完好，可以转移到克隆体中
- 黑井中濑的案例：被魔兽杀死后只剩灵魂宝石完好，被姐姐奈奈的愿望"让妹妹起死回生"完全复活

### 克隆体项目贡献者
- 黑井中濑对"白板"克隆体技术有重大贡献
- 普罗米修斯研究所管理层参与研发

### 未解决问题
- 克隆复活的成功率具体是多少？
- 复活后意识连续性如何保证？
- 灵魂宝石是否保存完整记忆？

## 情感抑制 (Emotional Suppression)

在军事行动中广泛使用的技术，旨在帮助魔法少女和指挥官在极端压力下保持冷静。

| 特性 | 描述 |
|------|------|
| **作用** | 降低或消除恐惧、悲伤等负面情绪，提升决策效率与战斗执行力。 |
| **副作用** | 长期使用可能导致情感淡漠。解除抑制（减压）时会产生剧烈的心理反弹，可能伴随呕吐、痛哭等身体与情绪反应。 |
| **应用范围** | 指挥官（如鄂温马克）用于维持战略定力；魔法少女战斗员用于应对惨烈的战场。 |
| **代偿机制** | 魔法少女往往在战斗后通过“宣泄”（如痛哭）来平衡长期抑制带来的心理压力。 |
