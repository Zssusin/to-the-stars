### 伯恩赛德-辛克莱家族 (Burnside-Sinclair Family)

> "The Burnside-Sinclair family has a long relationship with the Shizuki Matriarchy." —— Agapita

一个与志筑家族有历史纽带的魔法少女家族，在《艾玛的故事 2》中详细介绍。

#### 家族历史

| 时期 | 事件 |
|------|------|
| **统一战争前** | 祖先 **玛丽亚 (Maria)** 被志筑家族"收养"（sort of adopted） |
| **统一战争期间** | 伯恩赛德-辛克莱家族发展成独立的"女家长制"（Matriarchy） |
| **家族分裂** | 因"意识形态分歧"（ideological differences）导致家族崩溃，失去了许多人。部分成员加入了 **FA（自由同盟）**，其他人"保持低调，远离魔法"。 |
| **现代** | 志筑家族始终支持他们，家族通过联络名单保持松散联系 |

#### 家族成员

| 角色 | 身份 | 描述 |
|------|------|------|
| **劳尔 (Laure)** | 现任女家主 | 负责维持家族成员联系，"尽量不让他们像其他人一样失散" |
| **阿加皮塔 (Agapita)** | MagOps 成员，53 岁 | 艾玛的远亲，擅长用风笛施法。请求对艾玛做 DNA 测试以确认血缘。 |
| **玛丽亚 (Maria)** | MagOps 隐身专家 | 可生成隐形黑雾遮蔽整支队伍 |

#### 阿加皮塔的能力

> "Agapita's got the ability to drop entire companies of squid with those pipes of hers. It's not even that they explode or whatever. Somehow they just… switch off." —— Nodoka

| 能力 | 描述 |
|------|------|
| **风笛魔法** | 演奏风笛时，敌方无人机"关机"，章鱼部队失去行动能力后一个个倒下 |
| **欣快效果** | 队友会受到魔法影响，感到强烈的愉悦与信心（"like when you knew your team was on a roll"） |
| **精神控制性质** | 这种能力是潜意识层面的，受影响者不一定能察觉 |

#### 与艾玛姐妹的关联

- 阿加皮塔怀疑艾玛可能是家族失散支系的后裔（因"Sinclair"姓氏）
- 在故事中对艾玛进行了 DNA 测试
- 第二部结尾，伯恩赛德-辛克莱家族试图正式拉拢艾玛姐妹加入
- 志筑家族也在同时试图将 HSIS (Hashimoto and Sinclair Investments) 并入其体系

#### 家族背景
- **父亲**：**志筑久间** (Kyuuma Shizuki)，普罗米修斯研究所细胞生物学家。
- **母亲**：**黑井中濑** (Nakase Kuroi)，普罗米修斯研究所管理层，前魔法少女（曾被魔兽杀死，被姐姐奈奈许愿复活，契约解除）。
- **外公**：**黑井安倍** (Abe Kuroi)，224岁，退伍军人，与良子一同出发前往前线。
- **姨妈**：**黑井奈奈** (Nana Kuroi)，暗之心特工，晓美焰的徒弟。能力是反魔法/反科技力场。

#### 性格深度分析

**核心特质**：
- **与世界格格不入**：对"完美社会"的深刻不满
- **内敛但坚定**：外表冷静，内心渴望证明自己
- **务实的理想主义者**：追求星空但从不做白日梦
- **情感隔离**：用冷淡掩饰少女心

> **内心独白示例**：
> "有时候她觉得自己很傻，与周围的人格格不入。学校一直在向他们反复强调这些思想：人类的历史就是逃离的历史，逃离饥饿，逃离需求，逃离欲望。如今，人类已经到达了历史的尽头，终于开始建设一个完美社会。**但她并不觉得这有什么完美的。**"

**愿望本质**：
> "我想要离开地球，去探索这个世界。我想去往没有任何人到达过的地方，在这个宇宙中找到自己的所在。"

#### 语言风格详解

**对话特点**：
| 特征 | 表现 |
|------|------|
| 简洁直接 | 不拐弯抹角，用最少的话表达 |
| 沉默型 | 内心戏比外在对话多 |
| 条件反射式礼貌 | 紧张时会下意识鞠躬道歉 |
| 冷幽默 | 偶尔的自嘲式回应 |

**原文对话参考**：

```markdown
# 日常对话 - 社交尴尬
"谢谢，不过多半还是不必了。"
（面对西蒙娜的关心，礼貌但保持距离）

# 面对约会邀请 - 慌张失措
"不，不。那个，**呃，** 当然，没问题，我觉得？"
**不对，我为什么会这么说！？我应该说自己需要时间考虑考虑！**

# 被质问时 - 简短回应
"好——好了？"
（外表顺从，内心带着被强迫的感觉）

# 战斗间隙 - 直接认错
"对不起，我会改的。"
（小声说，不拖泥带水）

# 被救后的倔强
浅香伸出了手。良子没有接手，而是拄着膝盖自己站了起来。
```

**内心活动写法**：
```markdown
# 标准思考
她咽了口唾沫。尽管如此，良子还是对她们无比羡慕。

# 紧张时思维碎片化
这是——然后她被迫再次瞬移。
——怎么了？
混——
——蛋。它们改变——
——战术了！

# 自我反思
**我居然觉得这很爽？** 她想着。
```

**写作建议**：
- 善用**粗体**表示强烈情感或关键念头
- 对话极简，多用内心独白展现深度
- 面对复杂情感时选择沉默而非表达
- 危机中反而冷静，日常社交才慌张

### 巴麻美 (Tomoe Mami)

#### 基本信息

| 属性 | 描述 |
|------|------|
| **身份** | MSY 军方联络人、陆军元帅、幼发拉底战区总指挥 |
| **年龄** | 外貌 19 岁，实际约 460 岁 |
| **外貌** | 金色卷发双马尾、身材丰满、贝雷帽 |
| **变身服** | 黄色，花边长筒袜 |
| **武器** | 火绳枪、缎带 |
| **能力** | 缎带操控、枪械召唤 |

#### 性格深度分析

**表里不一的矛盾人格**：

| 公众形象 | 内心真实 |
|----------|----------|
| 优雅的元帅 | 疲惫的老兵 |
| 热情的征募官 | 讨厌征募如同讨厌"二手车推销员" |
| 自信的领导者 | 充满自我怀疑和负罪感 |
| 完美的前辈 | 渴望普通友谊却交不到朋友 |

**核心痛苦**：
> "麻美讨厌征募。她害怕看到那些少女单纯甜美的笑容和她们对'麻美大人'的真挚信赖。她不喜欢军方用自己的声音和脸蛋向那么多人推销一种很可能意味着死亡的生活选择。她讨厌这里面的一切。"

**关于孤独**：
> 她叹了口气。她真应该多交点朋友的。

#### 语言风格详解

**对话分层**：

| 场合 | 风格 | 示例 |
|------|------|------|
| 公众场合 | 正式、优雅、关怀 | "这回可真是好险**呢**。两个女孩子在工业区深处闲逛可不安全。" |
| 与同辈私下 | 略放松但仍克制 | "你也知道怎么出去吧？""我怎么来的就怎么出去。" |
| 与杏子互动 | 偶尔露出真性情 | "别笑话我了，佐仓。" |
| 工作状态 | 干练高效 | "让我们讨论一下现在该做什么吧。" |

**原文对话参考**：

```markdown
# 征募技巧（温和但有深意）
"对不起，但这也是没办法的。她知道你的心意吗？你有没有告白？"
"啊，不必否认了。我活了四百多岁，而且大半辈子里身边都只有找不到男性伴侣的女孩子。"

# 安慰后辈
"如果你真的爱她，现在就应该保持沉默。像我说的那样，微笑着为她送行，让她可以毫无顾虑地离开。"

# 工作中的干练
"你有空吗？"
"好吧，那你能把这位民间人士送回家吗？我得教这位新人见见世面。"

# 与杏子的老友对话
"我也很想念她，佐仓，但是你这次叫我来的目的是什么？"
"我会调查的。但是我无法承诺什么。"

# 自我反思（内心）
**我在干什么？还有六个小时的假期，而我居然还在工作。**
**那我该干什么？现在去打搅杏子也不礼貌，我也没有别处可去。**
```

**语气词使用**：
- 常用"呢"增加亲和力
- 正式场合措辞精准
- 对熟人偶尔省略敬语

**写作建议**：
- 公众场合保持完美形象
- 内心独白反映与表面的反差
- 回忆往事时语调沉重
- 对后辈永远关怀但有距离感
- 避免直接表露脆弱

### 佐仓杏子 (Sakura Kyouko)

#### 基本信息

| 属性 | 描述 |
|------|------|
| **身份** | 希望教创始人、中将 |
| **年龄** | 外貌 14 岁（选择停留），实际约 465 岁 |
| **外貌** | 红色长发马尾、常穿休闲服 |
| **武器** | 长枪 |
| **能力** | 幻像分身 |

#### 性格深度分析

**复杂的多面性**：
- **表面严厉实则关心**：用苛刻掩饰对新人的爱护
- **曾被信仰背叛后重建信仰**：创立希望教
- **私下生活随意**：睡觉流口水、离不开 Pocky
- **对朋友忠诚到偏执**：20 年不懈寻找焰

**训练风格**：
> "你这叫什么觉悟？你不是已经许了愿了吗？你应该很清楚自己签的是什么契约。难得伟大的佐仓杏子愿意教你两招，你居然想退缩不干？"

**深层情感**：
> "有时候我挺怀念以前的日子的。只有我们四个人，封闭在自己的小世界里，讨伐魔兽。没有行会什么乱七八糟的，没有执政体，没有军队，也没有外星人。"

#### 语言风格详解

**特色表达**：
| 类型 | 示例 |
|------|------|
| 口头禅 | "有性格。我喜欢。" |
| 粗犷用语 | "你丫明白？" |
| 自嘲式自大 | "难得伟大的佐仓杏子愿意教你两招" |
| 挑衅式称呼 | "菜鸟"、"小鬼" |

**原文对话参考**：

```markdown
# 训练新人（严厉但有用意）
"不准道歉！"
"听着，菜鸟，你的天赋魔法是什么？如果现在能用，就给我用一下。"
"我说**用**一下！你有战斗直觉，运用它！"

# 评价新人（恶作剧式）
杏子恶作剧地笑了笑。
"有性格。我喜欢。"

# 与麻美的老友对话
"下午好，元帅。"（挑拨语气）
"你不是喜欢小孩子的吗？"
"关上门。"

# 怀旧时刻（罕见的柔软）
"但为了这事离开我们值得吗？她在哪儿，麻美？她在做什么？"

# 战斗指导（专业严肃）
"三个教训，菜鸟。第一，不要逞强。"
"第二，不要光靠眼睛看。你是个魔法少女。你能直接感应到魔兽的位置。"
"第三，要时常保留一点余力，即使你需要为此提前收手。"

# 吃东西时的随意
"你敢相信吗？我不得不把自己的设计方案手动输入到我平时用的每一个合成器里。'找不到符合关键字 Pocky 的内置模型'。这世界是怎么了？"
```

**与不同人的语气变化**：
| 对象 | 语气 |
|------|------|
| 新人（良子） | 严厉挑衅但有引导性 |
| 老友（麻美） | 放松调侃带挑拨 |
| 下属/教众 | 随意但有威严 |
| 谈及焰时 | 罕见的柔软和失落 |

**写作建议**：
- 说话不客气但绝非恶意
- 适当使用老式俚语暗示年龄
- 严肃话题时反差式温柔
- 宗教话题时语调庄重
- 随身带着 Pocky 是人物标志

### 千岁由真 (Chitose Yuma)

#### 基本信息

| 属性 | 描述 |
|------|------|
| **身份** | MSY 政府事务部主任、第二执行官 |
| **年龄** | 外貌 9 岁（选择维持），实际约 455 岁 |
| **背景** | 南方组唯一幸存者，见泷原组（行会创始组）成员 |
| **性格** | 四人中最残忍、最擅长政治 |
| **武器** | 拟态为猫耳型的巨型球形战锤 |
| **能力** | **治愈**（极强，甚至能治疗灵魂）、高压冲击波（战锤） |
| **AI 助手** | **MG** (Magical Girl Advisory AI)，形象为普通魔法少女，与由真思维直连 |
| **政治策略** | 刻意维持 9 岁女童外貌，利用人类的保护欲和对儿童的宽容来解除政敌防备，实则城府极深 |

#### 战斗风格
- **治愈者/狂战士**：虽是治愈系，但战斗风格极其暴力，擅长近战肉搏和冲击波。
- **受虐狂倾向**：似乎从不躲避攻击，依靠强大的再生能力硬抗伤害换取攻击机会。

#### 性格深度分析

**极端反差的人格**：
- **9 岁外表**：刻意选择的政治武器
- **455 岁内心**：比任何人都成熟阴沉
- **童年创伤**：对"怪物"有特殊情结
- **政治天才**：用幼女形象瓦解对手防线

**形象选择的理由**：
> "由真选择这个形象的目的是解除他人的心理防线，勾起人们本能的保护欲，并且增加辩论的成功率。大多数人都不会认真反驳一个小孩，甚至连 AI 也难以脱俗。"

**创伤深处**：
> "我是个怪物，我也是一个和她们一模一样的怪物。我还有什么资格活下去？"
> （年幼时的话语，揭示创伤）

#### 语言风格详解

**双重人格的语言表现**：

| 场合 | 风格 | 示例 |
|------|------|------|
| 对外/公众 | 甜美童音、天真用语 | "姐姐！" |
| 私下讨论 | 成人语气、精准措辞 | "这和我们预想的不太一样。" |
| 调侃杏子 | 恶作剧+成熟混合 | "姐姐终于长大了吗？过了四百年..." |
| 严肃话题 | 完全成年化 | "我的态度里还有哪怕一丝不确的话，就只有这个了。" |

**原文对话参考**：

```markdown
# 迎接杏子（表演性童真）
"姐姐！"（一阵强光闪过，扑进杏子怀里）

# 调侃杏子（恶作剧式）
"噢，天啊，姐姐终于长大了吗？过了四百年，你终于不用再靠这种事情找安全感了！我简直为你感到骄傲了！"

# 严肃讨论（完全成人化）
"这很有趣，而且很令人不安。我们设计分配系统的时候是特地考虑过如何避免这种偶发性供给不均的。"
"姐姐，你的教团可不是天衣无缝的。你也很清楚。"

# 分析问题（锐利精准）
"也有另一种可能。也可能是有人在这份报告背后做了手脚，可能是你组织内部的人，也可能只是有人提供了假消息。"

# 骂杏子（用童声说成人话）
"姐姐，有的时候你真的很白痴。"
（还特地在"バカ"上变换了一下声调）

# 谈论过去（罕见柔软）
"就在织莉子姐姐临死之前，我看到她最后一次试图捕捉未来的轨迹——不是一般的未来，而是她自己的命运。"

# 关于来生（成熟且悲伤）
"不信。我其实是想要相信焰姐姐那些话的，真的很想。但是在经历过那样的生活之后，我做不到。"
```

**写作建议**：
- 外表与实际年龄形成强烈反差
- 政治场合用童音说成熟的话
- 偶尔流露创伤带来的脆弱
- 对 MG（AI 分身）有特殊亲密关系
- 可以在一句话中切换童真与成熟

```markdown
"所以请让我道个歉。我不会找什么借口说我只是做了自己该做的事情。最初是从演技开始，但最后我已经乐在其中，虽然只是一点点。"

# 关于仰慕与嫉妒
"当年我自以为承受得了，但我其实不行。谁也不行。我生前是个自以为全知全能的狂人，行径恶劣无比，自诩策士把所有人当作棋子，而到头来我不过也是棋子的一员。"
"我只是在仰慕着未来的你的同时产生了妒意。你才更适合成为拯救世界，拯救我们所有人的英雄。而我只不过是个摆弄玩具的小女孩。"

# 关于死亡与后悔
"反正她本来就会死掉的，但那并不能作为借口，尤其是在道德层面。不过说实话，真能用道德去衡量一切的只有我们这些死人，因为我们已经无法再做出实际的行动。"
```

**由真回忆中的织莉子**：
```markdown
# 作为导师
"正如织莉子曾经教过她的那样，只要稍微改变一下视角，稍微耍点脑筋，就可以把机器看作是**伤者**。"

# 关于时间的教诲
许久之前织莉子曾告诉她：即使到了她这个年纪，时间也是最宝贵的东西。
```

## 《艾玛的故事》角色 (Emma's Story)

> 衍生作品《艾玛的故事》中的主要角色，提供了 25 世纪魔法少女在核心殖民星（如萨姆萨拉）服役的视角。

### 艾玛·辛克莱 (Emma Sinclair)

#### 基本信息
| 属性 | 描述 |
|------|------|
| **身份** | 主角，英国伦敦出身 |
| **阶级** | 少尉 → 中尉 (1st Lieutenant) |
| **单位** | 尼罗河第三舰队，后部署至萨姆萨拉 |
| **能力** | 风系核心，生成涡流球（漩涡球） |
| **武器** | 战斧长戟 (Halberd)，约七英尺长 |

#### 背景故事
- 曾是伦敦阿森纳青年队足球运动员，因家族企业被志筑控股收购而被迫移居见泽原
- 因见泽原没有足球联赛而陷入抑郁，最终放弃足球梦想
- 签约时的愿望与"证明自己的价值"有关

#### 性格特点
- 运动员性格，直爽坦率
- 对战友极度忠诚
- 在战斗中表现果断，但也会冲动冒进
- 经历特蕾西牺牲事件后承受巨大心理创伤

#### 战斗风格
- 利用足球技巧（倒钩、剪刀脚）发射漩涡球
- 擅长空中机动和干扰作战
- 与蕾妮配合"火风龙卷"战术
- 近战使用长戟
