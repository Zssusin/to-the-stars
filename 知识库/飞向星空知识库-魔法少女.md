## 能力分类体系

> **来源**：作者读者问答 - "分类界限很快就变得模糊起来，某些类别包含了过多种魔法，因为MSY想让魔法更容易被分类。"

### 主要能力类型

| 类型 | 描述 | 代表人物 |
|------|------|----------|
| **远隔能力组** | 念力、透视、读心 | 田中唯（读心）、帕特里西亚（透视） |
| **元素操控** | 火、雷电、重力等自然力控制 | 鸣原亚纱美（重力） |
| **治愈系** | 治疗自己和他人的伤势 | 千岁由真、美树沙耶加 |
| **武器专精** | 特定武器的极致强化 | 佐仓杏子（长枪）、巴麻美（火绳枪） |
| **召唤类** | 召唤物体、生物或武器 | 巴麻美（枪械召唤） |
| **屏障类** | 生成防护屏障/结界 | 萨拉·凯森 |
| **空间系** | 传送、瞬移、虫洞 | 志筑良子、弥娜·古耶尔 |
| **幻象系** | 产生幻觉或分身 | 佐仓杏子（Rosso Fantasma） |
| **反魔法系** | 无效化/削弱其他魔法 | 西蒙娜·德尔·马戈（第73章明确提及）、黑井奈奈 |

### 特殊能力说明

**传送者 (Teleporter)**：
- 能力稀有且极具战术价值。
- MSY 对传送者实行特殊保护政策。
- 良子 200 公里的传送距离打破历史记录（此前最高 63 公里）。

**读心者 (Mind Reader)**：
- 常被安排到领导岗位（需判断真伪）
- MHD（心理卫生部）大量使用读心者建立监控网络
- 田中唯是行会创始人之一

**反魔法系**：
- 极其罕见的能力类型
- 黑井奈奈：可强迫其他MG解除变身，可反制二战后任何科技产品
- 西蒙娜·德尔·马戈：可消除/削弱其他魔法

**独特能力系统——伏特加魔法 (Motya)**：
> "Motya's abilities centered around distilled spirits. Bottles of vodka, and the ability to distill specific object properties into them."

| 伏特加类型 | 效果 |
|-----------|------|
| 毛巾干燥性 | 自动保持身体干燥 |
| 护甲硬度 | 大幅提升物理防御 |
| 治疗 | 加速伤口愈合，可治疗核辐射损伤 |

**能力发展核心原则**：
1. 需要大量努力和时间
2. 从小规模开始，逐步扩展
3. 学习相关物理知识有助于魔法控制
4. 魔力消耗需要持续监控（差点耗尽宝石的风险）


## 愿望与能力的关系

### 核心规律

> **来源**：作者讨论 - "并没有什么规则阻止你经过深思熟虑后许下一个复杂的愿望——但你得发自内心地希望它实现。"

**愿望→能力的转化**：
1. 能力与愿望的**本质意图**相关，而非字面表述
2. 愿望越**纯粹**、越**发自内心**，获得的能力越强
3. 过度思考或他人指点会**降低潜力**

### 愿望冲突处理

> "当人们许下一个愿望时，宇宙就会被推上一条道路，在这条道路上，人们所许下的愿望都是成立的，而那些与之矛盾的愿望却不复存在。"

**例子**：
- 许愿成为成功音乐家的少女，不必担心有人许愿阻止她成功
- 但无法阻止之后有人许愿让她的成功之路"以失败告终"

### 愿望类型举例

| 愿望类型 | 示例 | 获得能力 |
|----------|------|----------|
| **探索/离开** | 良子：想离开地球探索宇宙 | 虫洞传送 |
| **拯救他人** | 爱子：治愈男友白血病 | 治愈能力（间接实现：许愿让科学家发明新疗法） |
| **复活** | 黑井奈奈：让妹妹起死回生 | 反魔法/反科技力场（极罕见） |
| **保护** | 沙耶加原作：治愈恭介的手 | 强力治愈能力 |
| **信仰相关** | 杏子原作：让人们相信父亲 | 幻象能力 |

### 优化愿望的技巧

> **MSY做法**："推广那些许下优质愿望的文娱作品"

**优质愿望特点**：
- 比表面愿望更深远（如：治愈个人→发明新疗法帮助更多人）
- 但**不能由他人直接指点**（会导致潜力下降）
- 必须发自内心，而非理性计算

**反例**：
> "治愈能力并不真的是你内在渴望的一部分，但是当别人已经向你指出来这一点后，你就很难再回到之前的状态，发自内心的许下原本那个真实的心愿了……"

## 契约潜质规律

### 潜质来源

> **来源**：作者问答 - "很容易确定一个人具有潜力的可能性，至少从基因层面上是如此。"

**已知相关因素**：
- **情绪特征**：情绪化、易于移情、易于不满
- **遗传倾向**：某些家族契约率远高于平均值（如黑井家、志筑家）
- **神秘因素**：统计数据显示基因无法完全解释高潜力家族

### 潜质的消失

> "有时候潜力会消失……这种表面上的随机性确实让MSY感到困惑，孵化者说这与它们无关。"

**可能原因**：
1. 过度思考导致内心不再纯粹
2. 时间窗口过去（如情绪恢复平静）
3. 因果律的干预（丘比理论）

### 契约最佳时机

> **来源**：第一章丘比的话 - "最好是现在就立刻立下契约，以避免想太多而导致潜力消失"

**窗口期特点**：
- 情感激动时刻（如亲人濒死）
- 愿望纯粹且强烈
- 没有过多外界干扰

### 契约统计数据

| 项目 | 数值 |
|------|------|
| 契约率 | 约 1/20,000 人 |
| 平均契约年龄 | 14 岁 |
| 魔法少女总数（2460 年代） | 约 150-200 万 |
| 平均寿命（契约后） | 约 250 年 |

## 变身机制

### 变身与能力强度

> "超出某个界限的魔力必须在变身后才能使用。良子变身后才能传送，沙耶加不变身时治愈能力大幅减弱，杏子不能召唤屏障。"

**变身的意义**：
- 解锁完整魔法能力
- 召唤魔法武器
- 激活灵魂宝石护甲等装备

### 变身服外形来源

> "服装的外形与人类意识的原型库、文化因素，以及家庭都有关……除非女孩在契约时特别清楚自己变成什么样子，或者愿望对此有什么暗示。"

**历史变迁**：
- **古代**：战斗修女、圣女贞德式、巫师式
- **维多利亚时代后**：儿童时尚理念传入日本
- **MSY文化扩散后**：褶边裙装成为主流

### 变身服颜色规律

| 颜色 | 代表人物 |
|------|----------|
| 绿色 | 志筑良子 |
| 黄色 | 巴麻美 |
| 红色 | 佐仓杏子 |
| 紫色 | 黑井奈奈 |
| 白色 | 美国织莉子 |
| 黑色 | 晓美焰、沈小龙 |
