## 战争概述

统一战争是人类历史上规模最大、破坏最严重的冲突，跨越2163年至2230年，包含多次世界大战，最终以联合阵线(UF)的胜利和全球统一而告终。

### 战争性质
- **意识形态冲突**：精英资本主义(FA) vs 人民主权(UF)
- **武器等级**：常规武器 → 大规模杀伤性武器 → 核武器
- **人口损失**：战后人口不及战前一半
- **环境影响**：大片地区化为核荒漠，气候极度不稳定

## 对立双方

### 自由联盟 (Freedom Alliance, FA)

| 属性 | 描述 |
|------|------|
| 成立时间 | 2160年联合国解散后 |
| 意识形态 | "自由国家组成的防御组织，保护成员国不受共产主义、干涉主义、军国主义压迫者国家的侵略" |
| 统治阶层 | 超产阶级（hyperclass）—— AI时代拥有资本的精英阶层 |
| 特点 | 人口控制政策、残酷镇压起义、开发FA精锐部队 |
| 结局 | 2230年彻底灭亡 |

### 联合阵线 (United Front, UF)

| 属性 | 描述 |
|------|------|
| 成立时间 | 2160年 |
| 意识形态 | "将地球上国家的主权复归给为人民服务的政府，而非为私利攫取国家机构的既得利益精英阶层" |
| 核心概念 | "正当性声明" |
| 继承组织 | EDC（紧急防务评议会）→ 执政体(Governance) |

## 重大战役与事件

### 圣彼得堡大屠杀（2163年）

战争的导火索事件：
- FA超产阶级决定因反政府行为杀害城市一部分人口
- 全世界对此感到震惊
- MSY中FA亲和派蓄意背叛，导致大量新契约魔法少女死亡
- 直接导致MSY召开全体会议，投票越过《章程》通过最后通牒

### 千花运动（第四次世界大战期间）

"自由世界"组织策划的重大事件：
- 在中国境内爆发的大规模叛乱和骚乱
- 令捉襟见肘的UF政府猝不及防
- 国家大片区域长期失去控制
- 中国被迫召回绝大部分远征军，打乱联合阵线作战部署

**结果**：
- 促成UF在第四次世界大战中败北
- 但FA自身也差点覆灭
- 归来的远征军重组政府，肃清FA分子
- 中国此后成为UF中流砥柱

### 净化运动（Kiyomemasu，2218年后）

名称来源：日文"魂の世界は清めますが"（净化世界之魂）
- 周芷仪在巴基斯坦被推翻
- 退守喀喇昆仑山脉山区据点
- 最终山区据点被攻克
- FA精锐部队计划被发现

## MSY在战争中的角色

### 立场选择

> "MSY并没有被排除在这场动荡外。该组织的大多数成员都坚定地站在联合阵线一方，这反映了领导层和新加入的魔法少女的主张，并在很大程度上反映了魔法少女的基本天性。"

**选择UF的原因**：
1. 领导层与新魔法少女的主张
2. 魔法少女的基本天性
3. FA国家的暴行
4. FA人口减少政策威胁魔兽产生（MSY赖以生存的资源）

### 内部分裂

- 少数FA亲和派存在
- 圣彼得堡暴行后被揭露其蓄意背叛
- MSY全体会议通过最后通牒：
  - FA支持者必须立即转移到UF安全地点
  - 否则立即被捕并可能处决
- 大多数FA亲和派拒绝并留在原地
- 战后MSY成员在FA废墟中猎杀曾经的同胞

### 军事参与

> "统一战争的军事历史漫长且众所周知，但MSY于其中的参与则没那么有名。"

**MSY的行动**：
- 大规模扩展暗之心特工网络
- 执行民防任务
- 部署特种作战队（秘密陪同UF部队提供支持）
- 这种经验后来用于与章鱼的初次接触

### 组织变化

- 灵魂卫队被拉进来执行危险任务
- 暗之心引入新队伍
- 行会部队的起源可能与此有关
- 大部分权力在战争期间重新集中

## 战争遗产与影响

### 人类文明

| 领域 | 影响 |
|------|------|
| 人口 | 战后人口不及战前一半 |
| 环境 | 最不稳定的气候、核冬天、半数以上战前人类死亡 |
| 生物多样性 | 著名物种灭绝，仅动物园保存个体 |
| 技术 | 增强装置、植入芯片强制安装（2192年《全民战备法案》） |

### 政治制度

- EDC（紧急防务评议会）建立
- 2240年EDC解体，执政体成立
- 从2230年到2440年被视为"人类的黄金年代"

### 法律与伦理

1. **基因研究限制**：建立大量伦理委员会
2. **人类克隆禁忌**：因FA克隆士兵成为永恒禁忌
3. **AI权益**：目睹FA奴役AI的暴行后，赋予AI更大自主权
4. **EDC第224条决议**：一旦发现FA精锐部队立即射杀
5. **特别魔法少女征召法案**：参与战斗最低年龄规定

### 文化影响

- 见泷原在战争后保持惊人完好度和繁荣经济
- 统一战争后的经典电影如《阳光照耀格洛丽塔》
- 行会与媒体的关系成为前统一战争时代主旋律

### 对魔法少女的影响

- 远见会曾看到直白幻象（上次同等级幻象在统一战争时期）
- 某些战斗技能在统一战争开始时被正式传授
- 统一战争老兵需要特殊训练应对
