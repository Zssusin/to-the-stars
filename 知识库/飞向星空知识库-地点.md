## 地球核心区

### 见泷原市 (Mitakihara) - 人类文明的中心

| 属性 | 描述 |
|------|------|
| **重要性** | 故事起源地、MSY/行会发源地，25世纪人类文明的“中央处理器”。 |
| **特殊地位** | MSY"见泷原特区"、希望教总部。统一战争后保持了惊人的完好度和经济繁荣。 |
| **城市景观** | **42层以上的摩天楼林立**，空中交通管道密密麻麻如蜘蛛网般覆盖，远观如同精密的电路板。大部分区域因高楼遮挡**难以见到阳光和天空**。 |
| **气候控制** | 严格的气候管理系统将降雪量控制在“节日雪花”级别，确保城市运作不受干扰。 |
| **魔女街** | 老商业区的非官方称呼。云集行会各部（科学部、领导委员会等）和研究中心，是魔法少女社会的**社交与政治核心**。 |
| **太空电梯** | 位于新加坡东南海域的赤道**“海神”平台**，通过超音速喷气机（Scramjets）连接，4小时入轨。舷窗可见地平线弧度。 |

### 巴黎 (Paris) - 光明之城 (La Ville-Lumière)

> "巴黎之所以能有这样的地位，并不是因为什么纪念馆或者旅游地，而是因为什么样的生活都能在这里找到。" —— 克莱丽丝·凡·罗萨姆

#### 1. 城市结构与历史保护
*   **地下巴黎（Underground Paris）：** 为了保护巴黎具有悠久历史的**地标视线**，巴黎人拒绝在地面建造摩天大楼。作为替代，在深层地下挖掘了规模宏大的**“蜂巢式”都市**。
*   **艺术化布局：** 街道由艺术家和建筑师自底向上设计，每一层都有独特性格。
    *   **荧光洞窟站（la grotte éclatante）：** 装饰着水晶镶嵌图案，散发着奶白色荧光。
    *   **雪区（Neige）：** 维持在冰点以下并伴有人造降雪的特殊气象街区。
*   **地表局限：** 由于古迹保护，地表无法兴建现代化的高速交通管道，显得较为拥堵。

#### 2. 政治与情报中枢
*   **经济与军事中心：** 统一战争后成为 EDC 的核心，也是行会处理全球事务的关键节点。
*   **暗之心（Black Heart）基地：** 行会秘密情报组织的训练基地所在地，大部分特工在此完成初始培养。
*   **圣芭芭拉修会（Convent of Sainte-Barbe）：** 拥有两百年历史的深层地下修道院。配备了巨型干扰装置，可防范一切电子和魔法窃听，是行会高层的绝密会议室。

#### 3. 科学研究与文化
*   **娜塔莉·阿尼索特实验室（Natalie Anisette Laboratory）：** 全球知名的**瘴气（Miasma）**解析机构，提出了**“梦境世界假说”**。
*   **人类大档案馆（Great Archives of Humanity）：** 坐落于战时炸毁的空地上，展示人类文明、殖民史和精密微缩景观的顶级博物馆。
*   **迷途之城：** 被视为能反映每个人内心追求的“镜子”，吸引着无数失去方向的灵魂。

#### 4. 独特的生活体验
*   **前工业风：** 街道设计模仿 19 世纪欧洲风格，随处可见木质招牌和暗黄色路灯，内部却整合了尖端技术。
*   **昂贵的实体服务：** 巴黎仍保留了大量由人类（而非机器人）担任的店员与厨师，在 25 世纪被视为极度奢侈的体验。

> **巴黎比喻**：
> 25世纪的巴黎就像是一个**被装进精美地窖的古董首饰盒**：为了不惊动盒子表面那些脆弱而珍贵的历史花纹（地平线与古迹），人类在盒子下方挖掘了深邃的迷宫。它既是展示人类文明辉煌过往的博物馆，也是行会处理最黑暗秘密的深渊。

## 核心星球 (The Core Worlds)

> 第一波殖民浪潮（2260-2280年）建立，与地球构成正四面体分布，是人类文明最稳固的基石。

### 纽泰拉 (Nova Terra)
| 属性 | 描述 |
|------|------|
| **地位** | **人类第一个核心殖民地**，环境与地球最为契合，生态系统与人类适应性极佳。 |
| **因佩里亚城** | 首都。**绀碧色（深蓝绿色）天空**下，本土原生的巨大树木簇拥着摩天楼群，太阳撒下斑驳光影。 |
| **文化特征** | 相比于地球的钢铁森林，纽泰拉更倾向于使用**有机/木质材料**和花香装饰建筑。居民偏好穿着**鲜艳的彩色长袍**。 |
| **社会** | 拥有准执政体级别的自治权，管理 AI 为**泰拉诺瓦 (诺瓦)**。已接近与地球合并的阶段。 |
| **特产** | **C1 Aspera Cibum**（基因改造谷物，具有毛茸茸的红色叶片，是高档餐厅的珍贵食材）。 |

### 萨姆萨拉 (Samsara)
| 属性 | 描述 |
|------|------|
| **地位** | 第二个核心殖民地，被誉为“星际田园”。生态恢复力极强。 |
| **主要城市** | **赫尔辛堡 (Helsinberg)**：位于阿拉提亚 (Allatia) 大陆科拉西盆地 (Kolasi Basin) 西缘山谷。商业与金融中心，原为农工中转站。 |
| **地理特征** | **科拉西盆地 (Kolasi Basin)**：工业重镇，周围高山环绕。**阿拉提亚 (Allatia)**：大型大陆。 |
| **建筑风格** | 摩天楼通常约为 **300 层**（地球约为 800+ 层）。建筑通常由水泥基底加装消融层和复合聚合物。 |
| **社会状况** | 尽管由执政体管理，但仍存在明显的社会阶级分化与贫富差距，尤其是在快速扩张的工业城市中。 |
| **饮食文化** | **依菲特原鸽**（肉色偏绿但极鲜美，常配亮蓝色辣酱生食）；**C2 Ambigolimax Fractus**。 |
| **景观** | 拥有广阔的未开发大陆。曾在萨姆萨拉攻势中受外星人重点打击，后由鄂温马克力挽狂澜。 |
| **文化** | 相比于纽泰拉的繁忙，节奏较慢。拥有独特的萨姆萨拉口音（性别模糊感）。 |

### 其他相关星球
- **胡雷-5 (Hulei-5)**：在接触战争中被毁灭的行星。主要都市为**新台北 (New Taipei)**，曾以特色炸鸡和独特的建筑风格闻名，后被反物质武器彻底焚毁。
- **新阿穆尔 (New Amur)**：艾玛服役后期涉及的星系，第 6 尼罗河舰队在此通过“收割者”对抗战术发起反击。

### 欧普塔姆 (Optimum)
| 属性 | 描述 |
|------|------|
| **地位** | 第三个核心殖民地。晓美焰失踪前留下的最后明确行踪地。 |
| **物理环境** | 拥有**稠密的大气层**，栖息着巨大的原生鹰类和白鹅。适合享受飞行的乐趣。 |
| **文化特征** | 科研圣地。由于焰曾在此留下大量技术笔记，该行星学术风气极盛，拥有极高密度的大学与研究所。 |
| **战略** | 靠近前线，曾遭受外星闪现炮射程威胁。部署了极高规格的 TCF 验证节点。 |

## 殖民地与星系专题

### 1. 泰拉罗哈 (Terra Roja) —— 云端与飞翼之城
泰拉罗哈是一个典型的环境驱动文化的非法殖民地，其独特之处在于对人类形态的激进改造。
*   **物理环境：** 这是一个拥有极高密度大气的星球，地表压力大到人类无法生存，因此居民生活在大气层顶部的**悬浮都市**中。天空呈现出暗红色，云团连绵不绝。
*   **建筑风格：** 城市由巨大的**螺旋尖塔**组成，这些建筑利用轻质气体浮在空中，门窗直接开在半空中，完全不考虑步行者的需求。
*   **文化特征：** 早期开拓者为了适应高密度大气，通过基因工程在后背长出了**脊刺与连接终端**，用以操纵生物机械双翼和尾巴实现自力飞行。在当地文化中，飞行不仅是交通手段，更是社交核心（如空中婚礼和派对），情侣会购买羽饰装饰飞翼来宣布关系，这种“飞跃”被视为成年的标志。

### 2. 阿克戎 (Acheron) —— 硫磺与温泉之乡
阿克戎是一个以极度炎热和火山资源为特色的早期殖民星。
*   **物理环境：** 地质年轻，火山活动频繁。**天空呈现蓝绿色，太阳发红**，大气充斥着硫磺味和火山粉尘。星港气温常年高达**45摄氏度**。
*   **文化特征：** 最初由采矿AI“约翰·亨利”利用废矿井开发出了**温泉文化**（如带空调的冷水浴室）。当地文化带有浓厚的“边疆资本主义”色彩，拥有著名的“阿克戎魔鬼队”。

### 3. 俾斯麦 (Bismarck) —— 永夜与冰雪的堡垒
这是元帅鄂温马克的故乡，以严酷的气候考验着居民的意志。
*   **物理环境：** 该星球曾遭受巨型小行星撞击，导致自转轴严重倾斜，季风极度剧烈，季节交替时会产生超级风暴。在漫长的冬季，整个星球处于**极夜状态**，被皑皑白雪和坚冰覆盖。
*   **文化特征：** 这里的文化深受其早期保卫战历史的影响。在接触战争初期，当地民兵利用天气控制站制造暴风雪对抗外星人，这种**不屈的守卫精神**成为了该星的文化底色。小镇尼尔马琴以冰刀作为主要地面交通方式。

### 4. 尤里德米 (Eurydome) —— 重力研究的学术殿堂
尤里德米是核心星球欧普塔姆（Optimum）附近的卫星，是人类顶尖的物理科研中心，气氛清冷且极具学术美感。
*   **物理环境：** 一个**低重力星球**，非常适合开展高能物理实验。
*   **文化特征：** 核心文化围绕学术展开。最具代表性的建筑是“理论重力学院”（ITG），其主楼由 AI 设计，结构映射自**卡拉比–丘流形**，内部走廊极其曲折，装饰有数学美感的繁复石纹。
*   **塔利亚市 (Talia City)：** 星球上最大的城市（虽然按地球标准很小），分布着大量先进实验室。也是**志筑良子与鸣原亚纱美**同居并进行重力研究的地点。

### 5. 雷峰 (Leifeng) —— 飓风肆虐的海洋世界
*   **物理环境：** 这是一个全球性海洋星球，常年笼罩在时速500公里的**超级飓风**之下。海面上布满了红绿色带状的浮游生物。
*   **社会文化：** 人类居住在巨大的**海上浮动都市**中。

### 6. 叶尼塞 (Yenisei) —— 菌类覆盖的荒原
*   **物理环境：** 寒冷而荒凉的边境星球，地面覆盖着厚重的真菌菌丝体和真菌茎。气候极其恶劣，经常有飓风级别的风暴横扫平原。
*   **生活形态：** 居民主要居住在半地下的封闭式城镇中，地表遍布资源存储仓和温室穹顶。

### 7. 核心世界补充与波斯波利斯 (Persepolis)
*   **波斯波利斯：** **黄河战区**前线星。丘陵起伏，夕阳如血。米底 (Medea) 为第二大城市。
*   **阿波罗 (Apollo)：** 干燥沙漠行星，**大气呈现蓝紫色**，空气中带有砂岩被烘烤后的焦味。它是重要的军事和造船枢纽（波鲁克斯船坞）。
*   **纽泰拉 (Nova Terra)：** 第一个核心殖民星，环境与地球最像，但文化上更倾向于使用**有机/木质材料**和花香装饰建筑。

---

> **比喻理解**：
> 这些殖民地就像是人类文明在星际间散落的**试验田**。有的星球（如阿克戎和雷峰）是为了特定的**工业资源**而强行改造的硬核工地；有的（如泰拉罗哈）则是人类为了追求彻底的自由，不惜**打破自身生物边界**而建立的异类乌托邦；而像尤里德米这样的星球，则是将**数学与艺术**凝固在建筑中的科学圣殿。每一个世界都是对“人类身份”在不同极端环境下的一次重新定义。

## 外星领域

### 触手体 / 思裔领域 (Tentacle Domains)
- **范围**：**仙女座星系 (M31)** 棒状核心附近。
- **文明规模**：约 100 个相互独立的“触手体”组成，总人口超过**一万亿**。
- **资源**：每个触手体占据仙女座约 1% 资源（人类在银河系占比 < 0.1%）。
- **景观**：其原生环境多为温暖的海洋，但实验设施会设立在极端的高海拔山峰。
- **红链 (Red Chain)**：由一系列围绕红矮星建立的虫洞中继站组成，是连接银河系与仙女座的生命线。

---

> **类比理解**：
> 如果将25世纪的人类文明比作一台超级计算机，**见泷原**就是它的**中央处理器 (CPU)**；**核心三星球**是它的**稳定内存 (RAM)**；**工业殖民地**是它的**生产总线**；而**外星领域**则是那片尚未完全解析的、庞大且充满威胁的**外部互联网络**。
